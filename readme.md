# 陕西干部网络学院自动刷课工具

手动登录、选课后，本程序可以自动学完所有已选课程，如果发生进度条不走等情况，可自动重试。使用selenium模拟点击实现，一次只能登录一个账号。

# 兼容性

chromedriver.py使用了pywin32进行窗口控制，winget进行chrome安装。删除此部分代码后可以在linux运行。
UOS发行版已对此文件进行修改，代码与开源版本不同。

# 使用方法

0.注意设置操作系统的电源选项，阻止其自动休眠。
Win10/win11设置方法：开始-设置-系统-电源-屏幕和睡眠，将“插入电源时，闲置以下时间后将设备置于睡眠状态”选为“从不”
1.安装chrome浏览器。https://www.google.cn/chrome/
2.运行bin/main.py，（发行版运行main.exe），等待程序启动浏览器。
3.完成登录和选课。
4.输入要学习的课程序号。
5.等待程序运行。
程序开始显示播放进度后就可以开始进行其他工作，保持chrome和本程序在后台运行即可（可以最小化）。
程序运行时，如果需要使用浏览器，请使用其他浏览器，不要在播视频的这个chrome上操作。
软件正常运行时，会每10秒输出一个句号，出现其他情况说明软件已停止运行。
